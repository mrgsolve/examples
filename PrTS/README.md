To run the code, you will need to install these packages:


- `mrgsolve`
- `dplyr`
- `tidyr`
- `readr`
- `ggplot2`
- `parallel`
- `magrittr`
- `metrumrg`

To install `metrumrg`, try:
```r
install.packages("metrumrg", repos=c(options()$repos, "https://mrgsolve.github.io/pkg"), type="source")
```
